# Fuentes

Necesitas copiar **5 archivos TTF** a esta carpeta antes del primer `eas build`:

- `PlusJakartaSans-Regular.ttf`
- `PlusJakartaSans-Medium.ttf`
- `PlusJakartaSans-Bold.ttf`
- `Fraunces-Regular.ttf`
- `Fraunces-Bold.ttf`

## Descarga (2 min, gratis)

1. https://fonts.google.com/specimen/Plus+Jakarta+Sans → "Download family"
2. https://fonts.google.com/specimen/Fraunces → "Download family"

Dentro de cada ZIP, **entra a la subcarpeta `static/`** y copia los archivos con el peso explícito (no los `VariableFont_*`).

Los nombres deben quedar EXACTAMENTE como los listo arriba — si vienen con otro nombre (ej. `PlusJakartaSans-SemiBold.ttf`), renómbralo.
